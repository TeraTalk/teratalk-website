# TeraTalk website

Static marketing and research pages for **TeraTalk** — an AI-assisted speech therapy project (final-year research, SLIIT).

Live site (GitHub Pages): [teratalk.github.io/teratalk-website](https://teratalk.github.io/teratalk-website/)

## Contents

- Home, domain, milestones, documents, presentations, demo, architecture, about, and contact pages
- Built as static HTML with shared `style.css`

## Development

Open `index.html` in a browser, or serve the folder with any static file server so asset paths resolve correctly.

If the public URL is not `https://teratalk.github.io/teratalk-website/`, update `og:url`, `og:image`, `twitter:image`, and `link rel="canonical"` in each HTML file to match your deployment base URL.
